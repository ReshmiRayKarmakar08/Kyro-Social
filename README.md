## 🌟 About Kyro Social

Kyro Social is a professional, full-stack social media application offering a beautifully designed, responsive experience across all screen sizes. Whether you are browsing from your desktop browser or natively on your Android device, Kyro Social provides a seamless, high-performance environment for users to connect, share, and interact.

### ✨ Key Features

- **🌐 Responsive Web App:** A fully responsive frontend that adapts perfectly to desktop, tablet, and mobile browsers.
- **📱 Native Android App:** A dedicated, production-ready Android application built with React Native for fluid mobile performance.
- **🔐 Secure Authentication:** Robust user authentication with support for secure password management and session handling.
- **💬 Real-time Messaging:** Connect instantly with friends through the integrated real-time chat system powered by Socket.IO.
- **🎨 Modern UI/UX:** A stunning, themeable interface (including Dark Mode) built with premium design aesthetics.
- **📑 Post & Media Sharing:** Share your thoughts, images, and engage with content through likes, comments, and saves.

## 💻 Tech Stack

### Frontend (Web)
- **React.js** with Vite
- **Modern CSS** (Custom theme, Glassmorphism, Responsive UI)

### Mobile App (Android)
- **React Native CLI** (0.79.1)
- **Android Studio** native integrations
- **React Navigation** (Stacks, Tabs, Drawers)
- **AsyncStorage** for local persistence

### Backend & Database
- **Node.js & Express** 
- **MongoDB** implementation
- **Socket.IO** for real-time notifications and chat

## 🛠️ Getting Started (Local Development)

### 1. Clone the repository
```bash
git clone https://github.com/YOUR_USERNAME/Kyro-Social.git
cd Kyro-Social
```

### 2. Start the Backend server
```bash
cd backend
npm install
npm run dev
```

### 3. Start the Web Frontend
```bash
cd ../web-frontend
npm install
npm run dev
```

### 4. Run the Mobile App (Android)
```bash
cd ../mobile-app
npm install
npx react-native start
# In a new terminal tab:
npx react-native run-android
```
